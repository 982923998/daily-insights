# Output Format Templates

针对工具应用、开源项目、硬核论文和技术圈讨论的输出模板。

---

## 标准格式（默认）

## 模板选择指南

| 格式 | 适用场景 | 长度 | 详细程度 |
|------|----------|------|----------|
| **标准** | 日常默认 | 中等 | 关键点 + 来源链接 |
| **速览** | 时间有限、快速扫描 | 短 | 仅标题 + 一句话 |
| **深度** | 需要详细了解 | 长 | 技术分析 + 社区反应 |
| **论文专项** | 专注学术内容 | 中等 | 论文为主 |
| **工具专项** | 专注工具/项目 | 中等 | 工具为主 |

## 可复用的模板元素

这些元素可以按需添加到任意模板中：

**快速统计**
```markdown
## 📊 今日快报
- **工具新项目**: [X] 个
- **热门论文**: [X] 篇
- **技术圈热议**: [X] 个话题
- **有代码的论文**: [X]%
```

**阅读时间**
```markdown
**预计阅读时间**: [X] 分钟
```

**相关内容**
```markdown
**延伸阅读**: [相关链接]
```
