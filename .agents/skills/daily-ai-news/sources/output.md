# 输出格式

## Subcategory 参考

| subcategory | 来源 |
|-------------|------|
| `GitHub Trending` | github.com/trending |
| `HN Show HN` | news.ycombinator.com/show |
| `HuggingFace Spaces` | huggingface.co/spaces |
| `Product Hunt` | producthunt.com |
| `Major Release` | 各公司官博 + Twitter |
| `Twitter/X` | twitter.com / x.com |
| `Reddit` | reddit.com/r/LocalLLaMA |
| `AI for Science` | GitHub / HN / 专项搜索 |

---

## 标准格式（默认）

```markdown
# 🤖 AI 技术日报

**日期**: [当前日期，如 2026年2月23日]
**覆盖时段**: 最近 24 小时
**信息来源**: GitHub · HN · Reddit · Twitter/X · [X] 条内容

---

## 🔥 热门工具与开源项目

### [项目/工具名称]

**一句话**: [最简洁的功能描述]

**核心亮点**:
- [技术特点 1]
- [技术特点 2]
- [可用性/部署信息]

**为什么值得关注**: [1 句话说明实用价值]

📅 **来源**: [subcategory] • [发布日期]
🔗 **链接**: [原始链接]
💻 **GitHub/Demo**: [链接（如有）]

---

## 🗣️ 技术圈舆论

### [话题/讨论标题]

**核心观点**: [1-2 句话]

**技术社区在说什么**:
- [观点 1]
- [观点 2]

**值得关注的原因**: [1 句话]

📅 **来源**: [subcategory] • [日期]
🔗 **链接**: [URL（如有）]

---

## 🎯 今日重点

1. **最值得动手试的工具**: [1 句话]
2. **最值得关注的开源项目**: [1 句话]
3. **技术圈最热的讨论**: [1 句话]

---

**生成时间**: [时间戳]
```

## 速览格式

```markdown
# 🤖 AI 技术速览 · [日期]

## 🔥 热门工具与开源项目
• [项目名] `[subcategory]` — [一句话描述] 💻 [链接]

## 🗣️ 技术圈舆论
• [话题] `[subcategory]` — [核心观点一句话] 🔗 [链接]

---
**一句话总结**: [今日最值得关注的一件事]
```

## 深度格式

```markdown
## 🔥 热门工具与开源项目

### [项目名称] `[subcategory]`

**一句话**: [功能描述]

**技术细节**:
- [底层技术/模型/框架]
- [性能/规格参数]
- [与现有工具的对比]

**社区反应**: [HN/Reddit/Twitter 上的讨论]

**快速上手**:
\`\`\`bash
[安装/运行命令]
\`\`\`

📅 **来源**: [subcategory] • [日期]
🔗 **链接**: [原始链接]

---

## 🎯 今日趋势分析

1. **[趋势 1]**: [结合今日内容的分析]
2. **[趋势 2]**: [结合今日内容的分析]
```
